# SMARTDRUG
The Smart Drug Project uses Python for development, Machine Learning algorithms for prediction, and NLP techniques for processing user input. Libraries like Pandas, NumPy, and NLTK handle data tasks, Flask manages the backend, Streamlit or web tech builds the UI, and Pickle stores trained models efficiently.



# 💊 Smart Drug Recommendation System

## 📌 Overview

The Smart Drug Project is an AI-based system designed to analyze drug-related data and provide intelligent recommendations. It helps in predicting drug safety, identifying possible interactions, and assisting users with better healthcare decisions.

---

## 🚀 Technologies Used

* **Python** – Core programming language
* **Machine Learning** – Logistic Regression, Naive Bayes, Random Forest
* **NLP (Natural Language Processing)** – Tokenization, TF-IDF
* **Libraries** – Pandas, NumPy, NLTK / SpaCy
* **Backend** – Flask
* **Frontend** – Streamlit / HTML, CSS, JavaScript
* **Visualization** – Matplotlib / Plotly
* **Advanced Models (Optional)** – BERT (Hugging Face Transformers)

---

## ⚙️ Features

* Drug safety prediction
* Drug interaction analysis
* User input processing using NLP
* Interactive UI for easy usage
* Fast model loading using saved files

## ▶️ How to Run

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
2. Run the backend:

   ```
   python app.py
   ```
3. Open the frontend (Streamlit or browser)
